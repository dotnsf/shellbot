# Shell Bot for Node-RED

## Overview

Bot style shell for Node-RED.

## Setup

All you need is ...

- Node-RED runtime,

- and func-exec node( http://flows.nodered.org/node/node-red-contrib-func-exec ).

Then you can import shellbot_nodes.txt into your Node-RED canvas. Enjoy!
